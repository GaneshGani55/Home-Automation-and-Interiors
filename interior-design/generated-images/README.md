# Generated Images (`/banana` outputs)

All AI-generated interior renders go here. Use the **`/banana`** skill — do not call other image tools.

## Naming
`<room>-<style/mood>-v<N>.<ext>`

Examples:
- `living-warm-minimal-v1.png`
- `living-warm-minimal-v2.png` (iteration of the above)
- `master-bed-japandi-v1.png`
- `kitchen-handleless-modern-v1.png`
- `pooja-traditional-marble-v1.png`

## Notes per render
When a render is meaningful, drop a short note alongside it (same filename, `.md` extension) capturing:
- Prompt used
- What worked / what to change next
- Whether it was accepted as a direction (and if so, also log in [../../decisions/decision-log.md](../../decisions/decision-log.md))
